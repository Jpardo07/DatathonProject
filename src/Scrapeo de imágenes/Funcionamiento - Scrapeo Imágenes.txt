En este directorio se detalla el código empleado para la obtención de las imágenes utilizadas en el dashboard de visualización elaborado.

El dataset necesario para la reproducción del código se encuentra alojado en el apartado "General" del directorio de datasets proporcionado.

Para este caso, es necesario disponer de una instalación válida de la librería "Selenium" para Python3, así como un navegador Firefox y su webdriver "geckodriver".

La demora de ejecución de código, en una instancia de alta capacidad computacional "c2-standard-4", se alargó por más de 12 horas (La ejecución no se realizó de manera local).

------------------------------------------------------

El código de "Limpieza - Productos unicos" no requiere detalles adicionales, simplemente elabora un dataset con el nombre de los productos depurados para que Selenium pueda trabajar en "Scrapping - Imagenes".

Para reproducir este archivo, se requiere el dataset mencionado en el informe que se ubica, como se ha mencionado anteriormente, en el apartado "General" del directorio de datasets proporcionado.
